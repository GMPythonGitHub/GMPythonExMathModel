### Input lines: paste lines after starting program

``` python

```
[Case b]
``` python
40 -20
```
[Case c]
``` python
-80 40
```

## ExMM_D_01: geom_mltiples
２つの整数 $A$ と$B$ を考えます。  
次に示す４種類の $B$ の倍数を計算して, 表示して下さい。  
「$B$ の倍数のうち，$A$ 以下で最大のもの 」  
「$B$ の倍数のうち，$A$ より小さい最大のもの 」  
「$B$ の倍数のうち，$A$ 以上で最小のもの 」  
「$B$ の倍数のうち，$A$ より大きく最小のもの 」

### Input lines: paste lines after starting program
A B

[Case a]
``` python
15 5 
```
[Case b]
``` python
16 5
```
[Case c]
``` python
14 5
```
